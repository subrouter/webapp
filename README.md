Web Application Boilerplate
======
######v0.1

####Setup steps

Use composer to install web dependancies and point apache at the public folder for dev

###Server tech stack
* GNU Make
* Composer
* Silex

###Public tech stack
* backbone.js
* marionette.js
* require.js
* jquery.js
* sass
